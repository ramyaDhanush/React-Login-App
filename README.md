# React-Login-App
Create a react application for user login

1. Install NodeJS 
2. Verify Node & npm installation
` node -v` & `npm -v`
3. Checkout user-login folder
` cd user-login `
4. Install Login App using
` npm install` or `npm i` (shorthand)
5. To run the application
` npm start `
